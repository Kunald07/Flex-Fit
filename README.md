# FlexFit Landing Page

A modern and visually engaging fitness landing page built using pure HTML and CSS. FlexFit is designed to showcase a premium fitness brand with bold typography, dynamic layouts, fitness-focused imagery, and an attractive user interface.

## Preview

FlexFit presents a luxury fitness experience with:

* Modern card-based layout
* Responsive visual structure
* Fitness-themed branding
* Custom typography
* Font Awesome icons
* Interactive and engaging sections

## Features

* Clean and semantic HTML5 structure
* Modern CSS styling
* Custom Google Fonts integration
* Font Awesome icons
* Fitness dashboard cards
* Trainer showcase section
* Mobile app promotion section
* Location and branding components
* Premium landing page design

## Technologies Used

* HTML5
* CSS3
* Google Fonts
* Font Awesome

Through this project, I practiced:

* Typography hierarchy
* Visual storytelling through design
* Asset organization and project structuring
